# OculomotorStrategyToolkit

Toolbox for the extraction of the oculomotor metrics described in Maniglia, Visscher and Seitz (JOV, 2020)

Maniglia, M., Visscher, K. M., & Seitz, A. R. (2020). A method to characterize compensatory oculomotor strategies following simulated central vision loss. Journal of Vision, 20(9), 15. https://doi.org/10.1167/jov.20.9.15
